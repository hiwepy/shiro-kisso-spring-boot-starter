# shiro-kisso-spring-boot-starter
shiro starter for spring boot

### 说明


 > 基于 Shiro + Kisso 的单点登录 Spring Boot Starter 实现

### Maven

``` xml
	<artifactId>shiro-kisso-spring-boot-starter</artifactId>

```
